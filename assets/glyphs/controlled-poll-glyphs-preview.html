Construction — matches glyphs.tsx exactly

Each glyph below is the inner markup (the children of <G>). Paste as key: (p) => <G {...p}>…</G>. The two calendar variants reuse the existing calendar base path so they read as one family; the hourglass pair shares a frame; the shield/target/gauge corner feel is tuned to sit beside clock, soon and settings.

const G = ({ children, s, ...p }) => (
  <svg viewBox="0 0 24 24" width={s||24} height={s||24}
       fill="none" stroke="currentColor" strokeWidth="1.9"
       strokeLinecap="round" strokeLinejoin="round" {...p}>
    {children}
  </svg>
);
